# Home Controller

Controlador encargado de manejar la página `Home`.

## Estado
- `isLoggedIn`: verdadero si el usuario está identificado, falso en caso contrario.
- `itemsLoading`: veradero si se están cargando los datos, falso en caso contrario.
- `itemsData`: elementos que se encuentran en la raíz.
- `filteredData`: elementos de hoy.
- `archivedChildrenData`: elementos archivados de la raíz.
- `imageURL`: foto de perfil.

## Métodos
- `onNewGroup()`: crea un nuevo grupo.
- `onNewTask()`: crea una nueva tarea.
- `onNewNote()`: crea una nueva nota.
- `onNewBlock()`: crea un nuevo bloque de calednario.
- `onNavigateToProfile()`: navegar a la página de perfil.
- `onNavigateToCalendar()`: navegar a la página de calendario.
- `onNavigateToSearch()`: navegar a la página de búsqueda.
- `onNavigateToArchive()`: navegar a la página de archivo de la raíz.
