# Profile Controller

Controlador encargado de manejar la página `Profile`.

## Estado
- `data`: datos del perfil de usuario.
- `user`: datos del usuario.
- `imageURL`: foto de perfil.

## Métodos
- `onProfilePictureChange(value)`: cambia la foto de perfil del usuario.
- `onBack()`: vuelve a la vista anterior.
- `onChange(value)`: modifica el perfil.
- `onNavigateToTrash()`: navegar a la página de papelera del usuario.
- `validatePassword()`: valida la contraseña.
- `onPasswordChange()`: cambia la contraseña.
- `onLogOut()`: cierra la sesión del usuario.
- `onDelete()`: elimina la cuenta.
