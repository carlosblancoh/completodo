# Block Controller

Controlador encargado de manejar la vista `Block`.

## Props
- `id`: identificador del bloque de calendario que se quiere mostrar.

## Estado
- `data`: datos del bloque de calendario.
- `childrenData`: tareas asociadas al bloque.
- `groupData`: datos del grupo asociado al bloque de calendario.
