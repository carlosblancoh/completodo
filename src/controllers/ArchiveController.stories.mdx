# Archive Controller

Controlador encargado de manejar la página `Archive`.

## Parámetros URL
- `id`: identificador del grupo cuyos elementos archivados se quieren consultar.

## Estado
- `archivedItemsData`: elementos archivados del grupo.

## Métodos
- `onBack()`: vuelve a la vista anterior.
