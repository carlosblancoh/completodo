# Note Detail Controller

Controlador encargado de manejar la página `NoteDetail`.

## Parámetros URL
- `id`: identificador de la nota que se quiere mostrar.

## Estado
- `data`: datos de la nota.
- `parentData`: datos del grupo padre.
- `groupsData`: datos de todos los grupos.

## Métodos
- `onBack()`: vuelve a la vista anterior.
- `onChange(value)`: modifica la nota.
- `onDelete()`: elimina la nota.
